# myMoney Project

A personal money spending tracking app

Live Site URL: [myMoney](https://mymoney-project-da5ae.web.app/login)


## Built with

- HTML
- CSS
- React
- Firebase


## Features:

- Login
- Signup
- Realtime updates
- Add or delete personal spendings
- User can only view its personal spendings
